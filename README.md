# Super Store Sales Analytics Dashboard and Forecasting

## Overview
This project is an interactive sales dashboard built using Power BI to analyze business performance and generate insights from a retail dataset. It also includes a 15-day sales forecast based on historical data.

---

## Objective
To analyze sales trends, identify top-performing products and regions, and support data-driven decision-making.

---

## Business Problem
Businesses often struggle to track sales performance across regions, products, and time.  
This dashboard helps identify trends, high-performing products, and areas that require improvement.

---

## Tools Used
- Power BI  
- Microsoft Excel  
- Power Query  

---

## Dashboard Features
- Total Sales, Profit, and Quantity Overview  
- Monthly Sales and Profit Trends  
- Sales by Region and Customer Segment  
- Top Products and Categories  
- Sales by Shipping Mode  
- State-wise Sales Analysis  

---

## Data Cleaning and Transformation
- Removed missing and inconsistent values  
- Standardized date formats  
- Cleaned dataset using Power Query  
- Created calculated columns  

---

## Sales Forecasting
Used Power BI’s built-in forecasting feature to predict sales for the next 15 days based on historical trends.

---

## Key Insights
- West region has the highest sales  
- Office Supplies is the top-performing category  
- Sales increase during the last quarter (Q4)  
- A small number of products contribute to most of the revenue
